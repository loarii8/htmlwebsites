Web Development Tools

  1. Version Control
    1. Git
      1. Distributed version control system for tracking changes in source code.
    2. GitHub
      1. Web-based platform that provides Git repository hosting and team collaboration features.
  2. Code Editors
    1. VS Code
      1. Lightweight and powerful editor with extensions, Git integration, and debugging tools.
    2. Sublime Text
      1. Fast and customizable text editor known for its speed and plugin ecosystem.
  3. Browser Developer Tools
    1. Chrome DevTools
      1. Built-in tools for inspecting elements, debugging, analyzing performance, and testing responsive design.
    2. Firefox Developer Tools
      1. Alternative browser toolset for HTML/CSS/JS inspection, accessibility, and CSS grid visualization.
  4. Package Managers
    1. npm
      1. Manages Node.js dependencies and JavaScript libraries in web projects.
    2. Yarn
      1. Alternative to npm with better performance, caching, and deterministic builds.
  5. Build Tools
    1. Webpack
      1. Module bundler for JavaScript applications, compiles and optimizes assets.
    2. Vite
      1. Modern front-end build tool with fast dev server and optimized production builds.
