• Containerization
    1. Docker
      1. How it works: Platform for developing, shipping, and running applications in containers.
    2. Kubernetes
      1. How it works: Orchestrates the deployment, scaling, and management of containerized applications.
    3. Podman
      1. How it works: Daemon-less, rootless container management system that is compatible with Docker CLI.
    4. OpenShift
      1. How it works: A Kubernetes-based platform for container management with built-in CI/CD and security features.
    5. Docker Swarm
      1. How it works: Native Docker clustering and orchestration tool for managing multi-container deployments.
    6. Rancher
      1. How it works: A container management platform for orchestrating and managing Kubernetes clusters across environments.
    7. LXC/LXD
      1. How it works: Provides system-level containers that offer full Linux distributions within containers.
