Programming Languages and Technologies

  • Programming Languages
    • Python
      • How it works: Interpreted language, dynamically typed, great for data science and web development.
      • Popular Libraries: NumPy, Pandas, Flask, Django.
    • JavaScript
      • How it works: Interpreted language, primarily used for web development, runs in the browser.
      • Popular Frameworks: React, Angular, Vue.js.
    • Java
      • How it works: Compiled language, platform-independent due to the Java Virtual Machine (JVM).
      • Popular Frameworks: Spring, Hibernate.
    • C++
      • How it works: Compiled language, supports both procedural and object-oriented programming.
      • Common Uses: Game development, system/software development.
  • Web Technologies
    • HTML
      • How it works: Markup language used to create the structure of web pages.
    • CSS
      • How it works: Style sheet language used for describing the presentation of a document written in HTML.
    • Node.js
      • How it works: JavaScript runtime built on Chrome's V8 engine, allows server-side scripting.
  • Databases
    • SQL
      • How it works: Language used for managing and manipulating relational databases.
    • NoSQL
      • How it works: Non-relational databases that store data in a format other than tabular relations.
      • Popular Types: Document stores (MongoDB), Key-Value stores (Redis).
